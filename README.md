# OpenSauce  API

## Docs:
The documentation for the API can be found [here](https://docs.opensauce.uk), please note that everything is not documented or completed.

# Commands
Start app with nodemon
```
npm run dev
```
Start App
```
node . 
```
OR 
```
npm start
```
Lint: 
```
npm run lint
```
